Dimension 1
  • Click and drag a node to reposition it.
  • Click and hold a node in place to remove it. The start and end nodes (squares) can't be deleted.
  • Click and hold an edge to create a new node in the middle.
  • The live option will update the fractal as you drag nodes. It's limited to 5 iterations for performance.
  • The random option randomly flips each line segment vertically.
